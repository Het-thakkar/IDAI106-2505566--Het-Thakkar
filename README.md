##🧘‍♂️ MindfulWork – AI-Powered Stress Relief Pod App
#📌 Project Overview

MindfulWork is a wellness-based application designed to help reduce stress in workplaces using personalized experiences. The idea behind the app is to connect employees with smart stress relief pods that adjust their environment based on the user’s stress level and preferences. Instead of using general stress management techniques, this app focuses on creating a customized relaxation experience for each individual.

This project was created as part of the Artificial Intelligence (CRS) course under the Design Thinking for Innovation program at Udgam School for Children.

#🎯 Objective

The main objective of this project is to design a user-friendly solution that helps employees manage stress quickly and effectively without disturbing their workflow. The app uses AI-based ideas to suggest music, lighting, aroma, and meditation techniques to create a calm and focused state of mind.

#👥 Target Audience

This application is mainly designed for working professionals who experience stress during their daily routine. It is also useful for companies that want to improve employee well-being and productivity by providing better mental health support.

#🧠 Design Thinking Process

In the empathize and define stage, research was done to understand how employees experience stress at work. Common problems identified included heavy workloads, continuous meetings, and lack of proper relaxation spaces. Based on this research, multiple user personas were created to represent different types of employees. Empathy maps were also developed to understand what users think, feel, and do when they are stressed. From these insights, the problem statement was defined as:
“How might we provide instant and personalized stress relief for employees without interrupting their workflow?”

During the ideation stage, different ideas were brainstormed to solve this problem. The main focus was on personalization and simplicity. Features like MoodSync AI, guided breathing exercises, pod scheduling, stress tracking, and a post-session journal were finalized. Additional ideas such as light therapy, aroma selection, and relaxing music were also included to enhance the experience.

In the prototyping stage, a visual and interactive model of the app was created using Canva. The prototype includes important screens like the dashboard, AI recommendation section, pod control panel, meditation screen, and journal section. The goal was to make the app simple, calming, and easy to use.

In the testing stage, the prototype was shown to a few users such as classmates and friends. Their feedback helped identify what worked well and what needed improvement. Most users found the app easy to understand and useful, especially the meditation feature. Based on feedback, small improvements were made to simplify navigation and improve the overall design.

#⚙️ Features

The app includes a stress tracking dashboard that shows the user’s current stress level and weekly progress. One of the key features is MoodSync AI, which suggests personalized settings like ambient music, red light therapy, and calming aromas such as lavender and eucalyptus.

There is also a guided meditation feature that helps users relax using simple breathing techniques. The pod control panel allows users to adjust music, lighting, aroma, and session duration according to their preferences. After each session, users can write their thoughts in a journal and track how their stress level has changed.

#🛠️ Tools Used

The prototype for this project was created using Canva. The entire process followed the design thinking approach, focusing on understanding user needs and building a solution around them.

📂 Repository Structure

The project repository is organized into three main folders. The “docs” folder contains research work such as personas, empathy maps, and problem statements. The “wireframes” folder includes the UI designs and prototype screens. The “data” folder stores any survey responses or testing data used during the project.

🚀 Future Improvements

In the future, this app can be improved by integrating wearable devices to track stress in real time. Features like voice-based emotion detection and a fully functional mobile application can also be added to make the experience more advanced and realistic.
